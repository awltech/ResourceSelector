ResourceSelector by XA-Labs

The Resource Selector is a module that offers the possibility to select, through a user dialog box, a Resource from an Eclipse Project.

Authors

Maxence Vanbesien

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/XA-Labs/ResourceSelector

Finally, you can install the ResourceSelector project in Eclipse using this update-site: http://xa-labs.github.com/ResourceSelector/site/

get the source code on GitHub : XA-Labs/ResourceSelector
